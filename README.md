# React v16 测试项目

该项目基于 **React v16** 构建，使用 **Node.js v16** 进行开发，旨在进行相关测试。使用 **npm** 进行依赖安装。

## 项目仓库

项目仓库地址：[git@github.com:220529/react-16.git](git@github.com:220529/react-16.git)

## 环境要求

- **Node.js**：v16
- **npm**：v6 或更高版本

## 使用说明

### 1. 克隆项目仓库

```bash
git clone git@github.com:220529/react-16.git
cd react-16
```

```
pnpm install
```

```
npm run start
```
